# Nexus | Premium Project Management Dashboard

A high-end, responsive project management interface with a dark theme, integrated analytics, and server management.

## Features
- **Dashboard**: Real-time stats and projects by user.
- **Project Management**: CRUD operations for projects, grouped by owner.
- **Team Management**: Parallel server inventory for Team 1 and Team 2.
- **Security**: Password masking with toggle-to-reveal.
- **Persistence**: Data saved to browser `localStorage`.

## Technologies
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Lucide Icons

## Deployment
This project is ready to be deployed as a static site on Vercel, Netlify, or GitHub Pages.
